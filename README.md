# HR-Analytics-Dashboard
This Dashboard showcases key workforce metrics and insights that help organizations monitor employee performance, hiring, retention, and overall HR effectiveness. 

This project helped me strengthen my skills in data visualization, KPI analysis, and business storytelling using Power BI.

🛠 Tech Stack Used
Microsoft Power BI
Microsoft Excel
SQL for data querying & transformation
DAX for KPI calculations and MoM analysis
Data Modeling & Interactive Visualizations

📚 Skills Applied
KPI Dashboard Design
Business Requirement Analysis
Data Cleaning & Transformation
Data Visualization
DAX Measures
Analytical Storytelling
